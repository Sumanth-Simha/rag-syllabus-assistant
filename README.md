# RAG Syllabus Assistant

An AI-powered chatbot that helps students understand their syllabus using Retrieval-Augmented Generation (RAG).

##  Features
- Extracts syllabus EXACTLY as provided
- Provides detailed explanations like a teacher
- Module-based intelligent retrieval
- Built using Gemini + LangChain + ChromaDB

##  Tech Stack
- Python
- LangChain
- Google Gemini API
- HuggingFace Embeddings
- ChromaDB
- Streamlit
